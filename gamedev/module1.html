Step-by-Step Instructions:

  1. Install Godot: Download from godotengine.org
  2. Create New Project: Click "Create" in the project manager
  3. Name Your Project: "Year10_GameDev_[YourName]"
  4. Import Assets: Create an "assets" folder and import your downloaded graphics
  5. Explore Interface: Familiarize yourself with the different panels
