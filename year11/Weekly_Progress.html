8-Week Programming Course: Teacher Overview

Goal: Take absolute beginners to successful completion of AS92004 Simple Contact Book assessment

Course Overview

This document provides a high-level overview of the 10-week lesson plan designed to take absolute beginners to a point where they can successfully complete the AS92004 Simple Contact Book assessment.

Note: The course is designed as 8 core weeks plus 2 weeks for project development and refinement.

Weekly Progression

Week Topic Key Skills Learned Relation to Final Assessment
1 Intro to Programming & Data Types print(), input(), variables, strings, int(), type conversion Getting user input and understanding why it might need converting.
2 Conditional Logic & String Formatting if/elif/else, comparison operators, f-strings Building the menu and creating clean, readable output.
3 Introduction to Lists Creating lists, append(), pop(), remove(), accessing by index Storing the collection of contacts.
4 Iteration (Loops) while loops, for loops, break Creating the main menu loop and viewing all contacts.
5 Structuring Data (Dictionaries) Creating dictionaries, accessing values, .items(), del (Merit) Storing contact data in a well-structured way.
6 Error Handling try...except blocks to handle errors gracefully (Excellence) Making the program robust by handling invalid user input.
7 Functions & Documentation def, parameters, return, comments vs. docstrings (Excellence) Structuring code for flexibility and reusability.
8-10 Project Development & Refinement Applying all concepts, debugging, documentation, and final testing Completing the final assessment to a high standard over 3 weeks.

Assessment Grade Pathways

  • Achieved Level: Students complete Weeks 1-4 successfully and can implement basic functionality using sequence, selection, and iteration.
  • Merit Level: Students also master Week 5 (Dictionaries) to create well-structured data storage and demonstrate clear, logical code organization.
  • Excellence Level: Students additionally complete Weeks 6-7 (Error Handling & Functions) to create flexible, robust programs that handle edge cases gracefully.

Key Teaching Points

Weeks 1-4: Foundation Skills (Achieved Level)

  • Focus on core programming concepts that every student needs
  • Emphasize practical application and immediate feedback
  • Build confidence through incremental complexity
  • Ensure solid understanding before moving to advanced topics

Week 5: Data Structure (Merit Level)

  • Introduce dictionaries as a more sophisticated way to store related data
  • Compare list vs dictionary approaches for contact storage
  • Emphasize code readability and logical organization

Weeks 6-7: Advanced Concepts (Excellence Level)

  • Error handling: Making programs that don't crash
  • Functions: Code reusability and logical separation
  • Documentation: Professional coding practices
  • Testing: Systematic verification of program functionality

Weeks 8-10: Assessment Period

  • Students implement their contact book using learned concepts
  • Individual support based on target grade level
  • Focus on testing and documentation
  • Iterative improvement and debugging
