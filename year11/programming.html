Advanced Programming & AS92004 Assessment

Course Overview

Welcome to Year 11 Programming! This comprehensive course takes you from programming fundamentals to creating a robust Contact Book application. Master advanced programming concepts while working towards the AS92004 NCEA Level 1 assessment.

What You'll Master:

Advanced data types and variables with proper type conversion
Complex conditional logic with if/elif/else statements
Data structures: Lists and Dictionaries for organized data storage
Iteration techniques with while and for loops
Robust error handling using try...except blocks
Functions and modular programming with parameters and returns
Professional documentation and testing practices

Assessment Structure:

This course is designed around the AS92004 standard: "Create a simple computer program to meet a given brief." You'll build a Contact Book application that demonstrates programming competency at Achieved, Merit, or Excellence levels.

Learning Approach

This course follows a progressive learning model: Introduce ‚Üí Practice ‚Üí Apply ‚Üí Assess. Each week builds upon previous concepts, culminating in a comprehensive assessment project that demonstrates your programming competency.

The Year 11 program focuses on creating robust, well-documented programs that handle errors gracefully and solve real-world problems. By the end of this course, you'll have the skills needed for advanced programming courses and NCEA Level 2 Digital Technology.

NCEA Assessment Grades:

Achieved: Create a working program that meets basic requirements
Merit: Use appropriate data structures and demonstrate programming conventions
Excellence: Include robust error handling and comprehensive documentation
