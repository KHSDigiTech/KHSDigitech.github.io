What is a List?

A list is a data structure that holds an ordered sequence of items. Think of it like a numbered list of groceries or a column in a spreadsheet.

# An empty list
my_friends = []

# A list with initial items
shopping_list = ["apples", "bread", "milk"]

Adding to a List with .append()

.append() is a method that adds a new item to the very end of the list.

shopping_list = ["apples", "bread"]
shopping_list.append("eggs")
# Now, shopping_list is ["apples", "bread", "eggs"]

Accessing List Items by Index

Every item in a list has a position called an index. Indexes start at 0, not 1.

colors = ["red", "green", "blue"]
first_color = colors[0] # This gets "red"
second_color = colors[1] # This gets "green"

The number inside square brackets is the index.

Getting List Length with len()

len() is a function that tells you how many items are in a list.

colors = ["red", "green", "blue"]
num_of_colors = len(colors) # This would be 3

This is very useful for checking if a list is empty or for loops.

Modifying List Items

You can change an item at a specific index by assigning a new value to it.

colors = ["red", "green", "blue"]
colors[1] = "yellow" # Change item at index 1
# Now, colors is ["red", "yellow", "blue"]

Removing Items from Lists

Two common ways to remove items:
‚Ä¢ .pop(): Removes item at specific index (or last item)
‚Ä¢ .remove(): Removes first item matching the value

colors.pop(0) # Removes "red"
colors.remove("blue") # Removes "blue"
