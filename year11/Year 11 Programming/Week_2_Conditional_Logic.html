1. What is Conditional Logic?

Conditional logic allows your program to make decisions and execute different code paths based on certain conditions. Think of it like a flowchart where your program can take different routes depending on the situation.

Real-World Analogy:

Conditional logic is like making decisions in everyday life: "If it's raining, take an umbrella. If it's sunny, wear sunglasses. Otherwise, just go as you are." Your program can make similar decisions based on data.

2. The if Statement - Making Basic Decisions

An if statement checks if a condition is true. If it is, the indented code beneath it runs. If not, the code is skipped.

# Basic if statement
age = int(input("How old are you? "))

if age >= 18:
    print("You are an adult!")
    print("You can vote and drive.")

print("This line always runs.")

if Statement Breakdown:

  1. if - The keyword that starts the condition
  2. age >= 18 - The condition to check (must be True or False)
  3. : - Colon indicates the start of the code block
  4. Indentation - All code that belongs to the if must be indented (4 spaces)
  5. Condition result - If True, indented code runs; if False, it's skipped

3. Comparison Operators - Testing Relationships

Comparison operators let you compare values and return True or False.

Operator Meaning Example Result
== Equal to 5 == 5 True
!= Not equal to 5 != 3 True
< Less than 3 < 5 True
> Greater than 5 > 3 True
<= Less than or equal to 5 <= 5 True
>= Greater than or equal to 5 >= 3 True

‚ö†Ô∏è Common Mistake: = vs ==

  • = assigns a value to a variable: age = 18
  • == compares two values: age == 18

Using = instead of == in an if statement is a very common error!

# Examples of comparison operators
score = int(input("Enter your test score: "))

if score == 100:
    print("Perfect score!")

if score >= 90:
    print("Excellent work!")

if score < 50:
    print("Need to study more.")

if score != 0:
    print("At least you tried!")

4. The elif Statement - Multiple Conditions

elif (else if) lets you check multiple conditions in sequence. Python checks each condition in order and runs the first one that's true.

# Grade calculator using elif
score = int(input("Enter your score: "))

if score >= 90:
    grade = "A"
    print("Excellent!")
elif score >= 80:
    grade = "B"
    print("Good job!")
elif score >= 70:
    grade = "C"
    print("Satisfactory.")
elif score >= 60:
    grade = "D"
    print("Needs improvement.")
else:
    grade = "F"
    print("Failed. Study harder!")

print(f"Your grade is: {grade}")

                ‚Üì NO

5. The else Statement - The Default Option

else runs when none of the previous conditions are true. It's like a "catch-all" for any remaining cases.

# Simple password checker
password = input("Enter password: ")

if password == "secret123":
    print("Access granted!")
else:
    print("Access denied!")

# This always runs regardless of password
print("Security check complete.")

6. String Comparisons - Working with Text

You can compare strings just like numbers. Python compares them alphabetically and is case-sensitive.

# String comparison examples
name = input("What's your name? ")

if name == "Alice":
    print("Hi Alice!")
elif name == "Bob":
    print("Hello Bob!")
else:
    print(f"Nice to meet you, {name}!")

# Case-insensitive comparison
day = input("What day is it? ").lower()  # Convert to lowercase

if day == "saturday" or day == "sunday":
    print("It's the weekend!")
else:
    print("It's a weekday.")

7. Logical Operators - Combining Conditions

Logical operators let you combine multiple conditions in a single if statement.

Operator Meaning Example When True
and Both conditions must be true age >= 18 and age <= 65 Both conditions are true
or Either condition can be true day == "Saturday" or day == "Sunday" At least one condition is true
not Reverses the condition not (score < 50) The condition is false 
# Examples of logical operators
age = int(input("Enter your age: "))
has_license = input("Do you have a license? (yes/no): ").lower()

# Using 'and' - both conditions must be true
if age >= 16 and has_license == "yes":
    print("You can drive!")
else:
    print("You cannot drive yet.")

# Using 'or' - either condition can be true
weather = input("What's the weather? ").lower()
if weather == "rain" or weather == "snow":
    print("Take an umbrella!")

# Using 'not' - reverse the condition
if not (age < 13):  # Same as: if age >= 13:
    print("You can have social media.")

8. F-Strings - Modern String Formatting

F-strings make it easy to include variables and expressions directly in strings. They're cleaner and more efficient than older methods.

# F-string examples
name = input("What's your name? ")
age = int(input("How old are you? "))

# Basic f-string
print(f"Hello {name}, you are {age} years old.")

# F-strings with calculations
print(f"Next year you'll be {age + 1}.")
print(f"In 10 years you'll be {age + 10}.")

# F-strings with conditions
status = "adult" if age >= 18 else "minor"
print(f"You are considered a {status}.")

F-String vs Old Methods:

name = "Alice"
score = 95

# ‚ùå Old way - hard to read and error-prone
message1 = "Hello " + name + ", your score is " + str(score) + "%."

# ‚ùå Old way - using .format()
message2 = "Hello {}, your score is {}%.".format(name, score)

# ‚úÖ Modern way - clean and easy
message3 = f"Hello {name}, your score is {score}%."

9. The Critical Importance of Indentation

Python uses indentation (spaces) to group code together. This is not optional - incorrect indentation will cause your program to crash or behave unexpectedly.

‚ùå Common Indentation Errors:

# ERROR: No indentation
if age >= 18:
print("You can vote")  # This will cause an IndentationError

# ERROR: Inconsistent indentation
if age >= 18:
    print("You can vote")      # 4 spaces
        print("You can drive")    # 8 spaces - Error!

‚úÖ Correct Indentation:

# Correct: Use 4 spaces consistently
if age >= 18:
    print("You can vote")       # 4 spaces
    print("You can drive")      # 4 spaces
    if age >= 21:
        print("You can drink")   # 8 spaces (nested)
print("This is outside the if")  # No indentation

üö® Common Errors and How to Fix Them

Error 1: IndentationError

# Problematic code:
if age >= 18:
print("Adult")  # ERROR: Not indented

Fix: Always indent code inside if statements with 4 spaces:

if age >= 18:
    print("Adult")  # Correct: 4 spaces

Error 2: Using = instead of ==

# Problematic code:
if name = "Alice":  # ERROR: Assignment, not comparison
    print("Hi Alice")

Fix: Use == for comparisons:

if name == "Alice":  # Correct: Comparison
    print("Hi Alice")

Error 3: Comparing different data types

# Problematic code:
age = input("Age: ")    # This is a string!
if age >= 18:           # ERROR: Comparing string to number
    print("Adult")

Fix: Convert input to integer first:

age = int(input("Age: "))  # Convert to integer
if age >= 18:                # Now this works
    print("Adult")
