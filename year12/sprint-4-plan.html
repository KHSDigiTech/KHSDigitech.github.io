üìÖ Weekly Breakdown

Week 15: Game Feel & "Juice"
Focus: Make the game feel satisfying and responsive to play.

Tasks to Complete:

  • Add particle effects for key events (e.g., explosions, collecting items)
  • Implement screen shake for impactful moments

Week 16: Bug Hunting
Focus: Actively try to break your game and fix what you find.

Tasks to Complete:

  • Play through the game multiple times, trying unusual things
  • Create a list of all bugs found
  • Prioritize and fix the most critical bugs
  • Write weekly journal entry
Week 17: Final User Testing
Focus: Get a final round of feedback from a fresh set of eyes.

Tasks to Complete:

  • Prepare a feature-complete build of the game
  • Have 2-3 new people play the game
  • Log any final bugs or usability issues in the Feedback Log
  • Write weekly journal entry
Week 18: Final Sprint Reflection
Focus: Reflect on the sprint and prepare for final submission.

Tasks to Complete:

  • Complete the Sprint 4 Reflection
  • Prepare for final submission phase
  • Write weekly journal entry

üí° Development Tips

Bug Testing Strategy

  • Edge Cases: Try unusual player behaviors and input combinations
  • Platform Testing: Test on different devices if possible
  • Performance: Monitor frame rate and memory usage
  • User Flow: Ensure all menus and transitions work smoothly

Final Testing Tips

  • Fresh Eyes: Use testers who haven't played your game before
  • No Guidance: Let them discover the game naturally
  • Document Everything: Record both positive and negative feedback
  • Prioritize Fixes: Focus on game-breaking bugs first

‚ú® Polish & "Juice" Ideas

Game Feel Enhancements

  • Screen Shake: Small camera shake when player takes damage or destroys enemies
  • Particle Effects: Sparks, dust, explosion effects for key events
  • Sound Timing: Perfect synchronization of sound effects with visual events
  • Animation Easing: Smooth, natural movement curves instead of linear motion
  • UI Animations: Buttons that scale or glow when hovered
  • Visual Feedback: Health bars, score counters that animate smoothly
