üìã Task Requirements

Core Project Requirements

  • Game Development: Create a complete, playable game using Godot engine with multiple levels or significant gameplay depth
  • Advanced Programming: Implement complex programming concepts such as object-oriented design, data structures, algorithms, or AI behaviors
  • Design Process: Follow a structured design methodology with stakeholder engagement, user testing, and iterative refinement
  • Documentation: Maintain comprehensive documentation throughout the development process, including planning, progress tracking, and reflection
  • Technical Implementation: Demonstrate sophisticated understanding of programming principles and digital technologies
  • User-Centered Design: Show evidence of designing for specific users, gathering feedback, and improving based on testing
