Year 10 Game Development

Learn Godot Engine & AI-Assisted Programming

Program Overview

Welcome to the Year 10 Game Development program! This course is designed to teach you the fundamentals of game development using the Godot Engine, while also learning how to effectively use AI tools to help you code.

What You'll Learn:

Godot Engine interface and node system
Creating interactive game elements
How to use AI tools effectively for coding assistance
Building complete mini-games from scratch
Creating menus and connecting game scenes

Course Structure:

Each module is designed to be completed in one 55-minute class period. The focus is on hands-on learning through creating simple but complete games, with each module introducing new Godot nodes and concepts.

Course Modules

1️⃣

Module 1: Getting Started

Follow Brackeys' comprehensive Godot tutorial to understand the basics. Download essential graphics and set up your development environment.

Focus: Interface familiarization, project setup

 2️⃣

Module 2: Clicker Game

Create your first interactive game using Button and Label nodes. Learn the basics of UI elements and event handling.

Focus: UI nodes, signals, basic scripting

 3️⃣

Module 3: Character Movement

Build a simple character that can move around the screen using CharacterBody2D and Sprite2D nodes.

Focus: Character controllers, input handling

 4️⃣

Module 4: Collision Detective

Create a game where objects collide and interact using Area2D and RigidBody2D nodes.

Focus: Physics, collision detection

 5️⃣

Module 5: Simple Platformer

Build a basic platforming game using TileMap and StaticBody2D nodes for level design.

Focus: Tilemaps, level design, jumping mechanics

 6️⃣

Module 6: Maze Navigator

Create a top-down maze navigation game where players find their way from start to exit using a dedicated TopDownController.

Focus: 8-directional movement, maze design, goal detection

 7️⃣

Module 7: Simple Shooter

Build a basic shooting game using instantiation and Timer nodes to create projectiles.

Focus: Object instantiation, timers, projectile physics

 8️⃣

Module 8: Vehicle Controller

Build a realistic car racing game using RigidBody2D for authentic vehicle physics with acceleration, steering, and drift mechanics.

Focus: Realistic physics, vehicle dynamics, racing tracks

 9️⃣

Module 9: Menu Master

Create a main menu that connects all your previous games together into one cohesive project.

Focus: Scene management, UI design, project organization

Using AI for Coding

Throughout this course, you'll learn how to effectively use AI tools like ChatGPT, Claude, or GitHub Copilot to help with your coding. Each module includes specific prompts and techniques for getting the most out of AI assistance while still learning the fundamentals yourself.
