What are Lists?

A list is a collection that can hold multiple values in a single variable. Think of it as a container that can hold many items, like a shopping list or a list of students in a class.

Real-World Example

Instead of creating separate variables for each student (student1 = "Alice", student2 = "Bob", etc.), you can create one list: students = ["Alice", "Bob", "Charlie", "Diana"]

Creating and Using Lists

Basic List Creation

# Creating lists
fruits = ["apple", "banana", "orange", "grape"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, 3.14, True]
empty_list = []

print("Fruits:", fruits)
print("Numbers:", numbers)
print("Mixed:", mixed)
Fruits: ['apple', 'banana', 'orange', 'grape'] Numbers: [1, 2, 3, 4, 5] Mixed: ['hello', 42, 3.14, True]

Accessing List Items (Indexing)

colors = ["red", "green", "blue", "yellow"]

print("First color:", colors[0])    # Index 0 is the first item
print("Second color:", colors[1])   # Index 1 is the second item
print("Last color:", colors[-1])    # Index -1 is the last item

# Get the length of the list
print("Number of colors:", len(colors))
First color: red Second color: green Last color: yellow Number of colors: 4

List Methods

Lists come with built-in methods to modify them:

append()

Add item to end

list.append("new")

insert()

Add item at position

list.insert(1, "new")

remove()

Remove first matching item

list.remove("item")

pop()

Remove and return last item

item = list.pop()

Introduction to Functions

Functions are reusable blocks of code that perform specific tasks. Instead of writing the same code multiple times, you can create a function and call it whenever needed.

Why Use Functions?

  • Avoid repeating the same code
  • Make code easier to read and understand
  • Easy to fix bugs in one place
  • Break complex problems into smaller parts

Function Anatomy

def function_name(parameters):
    """Optional description of what the function does"""
    code to execute
    return value # Optional

Function Examples

Example 1: Simple Function with No Parameters

def greet():
    """Prints a greeting message"""
    print("Hello! Welcome to Python programming!")
    print("Hope you're having a great day!")

# Call the function
greet()
print("Let's learn about functions!")
greet()  # We can call it multiple times
Hello! Welcome to Python programming! Hope you're having a great day! Let's learn about functions! Hello! Welcome to Python programming! Hope you're having a great day!

Example 2: Function with Parameters

def greet_person(name, age):
    """Greets a person with their name and age"""
    print(f"Hello {name}!")
    print(f"I see you are {age} years old.")
    print("Nice to meet you!")

# Call the function with different arguments
greet_person("Alice", 16)
print()
greet_person("Bob", 15)
Hello Alice! I see you are 16 years old. Nice to meet you! Hello Bob! I see you are 15 years old. Nice to meet you!

Example 3: Function that Returns a Value

def calculate_area(length, width):
    """Calculate the area of a rectangle"""
    area = length * width
    return area

def calculate_circle_area(radius):
    """Calculate the area of a circle"""
    pi = 3.14159
    area = pi * radius * radius
    return area

# Use the functions
rect_area = calculate_area(10, 5)
print(f"Rectangle area: {rect_area}")

circle_area = calculate_circle_area(7)
print(f"Circle area: {circle_area}")

# You can use the returned value directly
print(f"Total area: {calculate_area(8, 6) + calculate_circle_area(3)}")
Rectangle area: 50 Circle area: 153.93791 Total area: 76.27433

Example 4: Function that Works with Lists

def find_average(numbers):
    """Calculate the average of a list of numbers"""
    if len(numbers) == 0:
        return 0
    
    total = 0
    for number in numbers:
        total += number
    
    average = total / len(numbers)
    return average

def find_highest(numbers):
    """Find the highest number in a list"""
    if len(numbers) == 0:
        return None
    
    highest = numbers[0]
    for number in numbers:
        if number > highest:
            highest = number
    
    return highest

# Test the functions
test_scores = [85, 92, 78, 96, 88, 91]

avg = find_average(test_scores)
high = find_highest(test_scores)

print(f"Test scores: {test_scores}")
print(f"Average: {avg}")
print(f"Highest: {high}")
Test scores: [85, 92, 78, 96, 88, 91] Average: 88.33333333333333 Highest: 96

Example 5: Complete Program Using Lists and Functions

def add_student(students, name, grade):
    """Add a student to the list"""
    student = {"name": name, "grade": grade}
    students.append(student)
    print(f"Added {name} with grade {grade}")

def show_all_students(students):
    """Display all students and their grades"""
    if len(students) == 0:
        print("No students in the list.")
        return
    
    print("\nAll Students:")
    print("-" * 20)
    for i, student in enumerate(students, 1):
        print(f"{i}. {student['name']}: {student['grade']}")

def calculate_class_average(students):
    """Calculate the class average grade"""
    if len(students) == 0:
        return 0
    
    total = 0
    for student in students:
        total += student["grade"]
    
    return total / len(students)

# Main program
class_list = []

add_student(class_list, "Alice", 92)
add_student(class_list, "Bob", 87)
add_student(class_list, "Charlie", 95)

show_all_students(class_list)

avg = calculate_class_average(class_list)
print(f"\nClass average: {avg:.1f}")
Added Alice with grade 92 Added Bob with grade 87 Added Charlie with grade 95 All Students: -------------------- 1. Alice: 92 2. Bob: 87 3. Charlie: 95 Class average: 91.3

Common Mistakes to Avoid

❌ Don't do this:

# List indexing errors
colors = ["red", "green", "blue"]
print(colors[3])           # Error: index out of range

# Function definition errors
def my_function()          # Missing colon
    print("Hello")

def my_function():
print("Hello")             # Missing indentation

# Forgetting to return
def add_numbers(a, b):
    result = a + b         # Missing return statement

✅ Do this instead:

# Safe list access
colors = ["red", "green", "blue"]
if len(colors) > 3:
    print(colors[3])
else:
    print("Index out of range")

# Correct function syntax
def my_function():         # Don't forget the colon
    print("Hello")         # Proper indentation

# Always return when needed
def add_numbers(a, b):
    result = a + b
    return result          # Return the result

Key Takeaways

  • Lists store multiple values in one variable using square brackets []
  • List indexing starts at 0 (first item is list[0])
  • Use append(), insert(), remove(), and pop() to modify lists
  • Functions help organize code and avoid repetition
  • Use def to create functions, and return to give back values
  • Parameters let you pass information into functions
  • Combine lists and functions to create powerful programs
