What are If Statements?

If statements allow your program to make decisions. They check if something is true, and if it is, they run specific code. Think of it like a decision tree - "If this is true, then do this."

Real-World Example

In real life: "If it's raining, then I'll bring an umbrella." In programming: "If the user's age is 18 or older, then they can vote."

Decision Flow

Question → True? → Do this action

Question → False? → Skip the action (or do something else)

Basic If Statement Syntax

Here's the basic structure of an if statement:

if condition:
    # Code to run if condition is True
    print("This runs if the condition is true")

⚠️ Indentation is Critical!

Python uses indentation (spaces or tabs) to group code together. All code that should run when the if statement is true must be indented by the same amount (usually 4 spaces).

Comparison Operators

These operators let you compare values and return True or False:

Equal to

==

5 == 5 → True

Not equal to

!=

5 != 3 → True

Less than

<

3 < 5 → True

Greater than

>

7 > 4 → True

Less than or equal

<=

5 <= 5 → True

Greater than or equal

>=

6 >= 4 → True

age = int(input("How old are you? "))

if age >= 18:
    print("You are an adult!")
    print("You can vote!")

print("This line always runs")

Example 5: Multiple Conditions with Logical Operators

age = int(input("Enter your age: "))
has_license = input("Do you have a driver's license? (yes/no): ").lower()

if age >= 16 and has_license == "yes":
    print("You can drive!")
elif age >= 16 and has_license == "no":
    print("You're old enough, but you need a license!")
elif age < 16 and has_license == "yes":
    print("How do you have a license? You're too young!")
else:
    print("You're too young to drive.")
Enter your age: 17 Do you have a driver's license? (yes/no): yes You can drive!

Example 6: Number Guessing Game

import random

secret_number = random.randint(1, 10)
guess = int(input("Guess a number between 1 and 10: "))

if guess == secret_number:
    print("Congratulations! You guessed it!")
    print("The number was", secret_number)
elif guess < secret_number:
    print("Too low! The number was", secret_number)
else:
    print("Too high! The number was", secret_number)

Logical Operators

Combining Conditions:

  • and - Both conditions must be true
  • or - At least one condition must be true
  • not - Reverses the condition (True becomes False)
# AND example
if age >= 13 and age <= 19:
    print("You're a teenager")

# OR example
if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")

# NOT example
if not is_raining:
    print("No umbrella needed")

Common Mistakes to Avoid

❌ Don't do this:

if age = 18:           # Use == not = for comparison
    print("Adult")
    
if age >= 18:
print("Adult")         # Missing indentation

if 15 < age < 20       # Missing colon
    print("Teen")

✅ Do this instead:

if age == 18:          # == for comparison
    print("Adult")
    
if age >= 18:
    print("Adult")     # Proper indentation

if 15 < age < 20:      # Don't forget the colon
    print("Teen")

Key Takeaways

  • If statements let your programs make decisions based on conditions
  • Use == to check if values are equal (not =, which assigns values)
  • elif lets you check multiple conditions in order
  • else runs when none of the previous conditions are true
  • Indentation is crucial - all code in an if block must be indented the same amount
  • You can combine conditions using and, or, and not
