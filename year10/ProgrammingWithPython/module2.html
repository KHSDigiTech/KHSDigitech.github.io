Week 2 - Storing and Working with Data

Learning Objectives

By the end of this module, you will be able to:

  • Create and assign values to variables
  • Understand the four main data types: string, integer, float, and boolean
  • Use variables in print statements and calculations
  • Follow Python naming conventions for variables
  • Change variable values and understand variable reassignment

What are Variables?

A variable is like a labeled box that stores information. You can put different types of information in the box, and you can use the label (variable name) to get the information back later.

Real-World Analogy

Think of variables like labeled containers in your kitchen. You might have a jar labeled "sugar" that contains sugar, and a jar labeled "flour" that contains flour. In programming, you might have a variable called "age" that contains the number 15.

Creating Variables

To create a variable, you use the assignment operator =. The variable name goes on the left, and the value goes on the right.

name = "Sarah" age = 15 height = 5.6 is_student = True

Variable Naming Rules:

  • Must start with a letter or underscore (not a number)
  • Can contain letters, numbers, and underscores
  • Cannot contain spaces or special characters
  • Case-sensitive (age and Age are different)
  • Cannot use Python keywords (like print, if, etc.)

Python Data Types

Python has several data types, but we'll focus on the four most important ones:

String (str)

Text data

"Hello"

Integer (int)

Whole numbers

42

Float (float)

Decimal numbers

3.14

Boolean (bool)

True or False

True

Detailed Examples

Example 1: Creating and Using String Variables

first_name = "John" last_name = "Smith" school = "King's High School" print("Student name:", first_name, last_name) print("School:", school)
Student name: John Smith School: King's High School

Example 2: Working with Numbers

age = 16 height = 1.75 weight = 65.5 print("Age:", age, "years old") print("Height:", height, "meters") print("Weight:", weight, "kg") # You can do math with number variables next_year_age = age + 1 print("Next year I will be", next_year_age)
Age: 16 years old Height: 1.75 meters Weight: 65.5 kg Next year I will be 17

Example 3: Boolean Variables

is_teenager = True has_license = False likes_pizza = True print("Is a teenager:", is_teenager) print("Has driver's license:", has_license) print("Likes pizza:", likes_pizza)
Is a teenager: True Has driver's license: False Likes pizza: True

Example 4: Changing Variable Values

score = 0 print("Starting score:", score) score = 10 print("After first level:", score) score = score + 5 # Add 5 to the current score print("After bonus:", score) score = 100 print("Final score:", score)
Starting score: 0 After first level: 10 After bonus: 15 Final score: 100

Example 5: Using Variables in Calculations

length = 10 width = 5 height = 3 area = length * width volume = length * width * height perimeter = 2 * (length + width) print("Rectangle area:", area) print("Box volume:", volume) print("Rectangle perimeter:", perimeter)
Rectangle area: 50 Box volume: 150 Rectangle perimeter: 30

Mathematical Operators

+ (addition), - (subtraction), * (multiplication), / (division)

Practice Tasks

Complete these 10 tasks to practice working with variables:

Task 1: Personal Information

Create variables for your first name, last name, and age. Print them in a sentence like: "My name is [first] [last] and I am [age] years old."

Task 2: Favorite Things

Create string variables for your favorite color, food, and movie. Print each one with a descriptive message.

Task 3: School Subjects

Create variables for three school subjects and their grades (as numbers). Print each subject and grade.

Task 4: Simple Calculator

Create two number variables with values 15 and 7. Calculate and print their sum, difference, product, and quotient.

Task 5: Rectangle Properties

Create variables for a rectangle's length (8) and width (5). Calculate and print the area and perimeter.

Task 6: Boolean Practice

Create three boolean variables about yourself (like is_tall, likes_sports, has_pets). Print each with a descriptive message.

Task 7: Variable Reassignment

Create a variable called "score" with value 0. Then change it to 25, then to 50, printing the value each time with a message.

Task 8: Temperature Conversion

Create a variable for temperature in Celsius (25). Calculate and store the Fahrenheit equivalent (F = C * 9/5 + 32). Print both temperatures.

Task 9: Circle Properties

Create a variable for a circle's radius (7). Calculate the area (π * r²) and circumference (2 * π * r). Use 3.14159 for π.

Task 10: Student Profile

Create a complete student profile using variables: name, age, grade level, favorite subject, GPA (as a decimal), and whether they play sports (boolean). Print all information in a nicely formatted way.

Common Mistakes to Avoid

❌ Don't do this:

2age = 15 # Can't start with number my age = 15 # No spaces allowed my-age = 15 # No hyphens allowed Age = age # These are different variables!

✅ Do this instead:

age2 = 15 # Numbers at end are OK my_age = 15 # Use underscores for spaces my_age = 15 # Use underscores for word separation age = 15 # Be consistent with capitalization

Key Takeaways

  • Variables store data that can be used later in your program
  • Python has four main data types: string, integer, float, and boolean
  • Use descriptive variable names that make your code easy to understand
  • You can change a variable's value anytime by assigning a new value
  • Variables can be used in calculations and print statements
