  • Convert strings to integers using int()
  • Convert strings to floats using float()
  • Convert other data types to strings using str()
  • Create interactive programs that respond to user input

The input() Function

The input() function allows your program to pause and wait for the user to type something. Whatever the user types becomes the value that input() returns.

How input() Works

When Python reaches an input() statement, it displays a prompt (if you provide one), waits for the user to type something and press Enter, then continues with that text as a string value.

Basic Input Syntax

Here's the simplest way to get input from a user:

name = input("What is your name? ")
print("Hello,", name)
What is your name? Sarah Hello, Sarah

⚠️ Important: input() Always Returns a String

No matter what the user types - even if it looks like a number - input() always gives you a string. If you want to use it as a number, you need to convert it!

Type Conversion Functions

Python provides built-in functions to convert between different data types:

int()

Converts to integer (whole number)

int("25") → 25

float()

Converts to decimal number

float("3.14") → 3.14

str()

Converts to string (text)

str(42) → "42"

Detailed Examples

Example 1: Getting and Using Text Input

first_name = input("Enter your first name: ")
last_name = input("Enter your last name: ")
favorite_color = input("What's your favorite color? ")

print("Hi", first_name, last_name)
print("I like", favorite_color, "too!")
Enter your first name: Alex Enter your last name: Johnson What's your favorite color? blue Hi Alex Johnson I like blue too!

Example 2: The Problem with Number Input (Without Conversion)

# This WON'T work as expected!
age = input("How old are you? ")
next_year = age + 1  # This will cause an error!
print("Next year you'll be", next_year)
TypeError: can only concatenate str (not "int") to str

Example 3: Correct Way - Converting Input to Numbers

# Convert string input to integer
age_text = input("How old are you? ")
age = int(age_text)
next_year = age + 1

print("You are", age, "years old")
print("Next year you'll be", next_year)
How old are you? 16 You are 16 years old Next year you'll be 17

Example 4: Shorthand - Converting Input Directly

# You can convert input directly in one line
age = int(input("How old are you? "))
height = float(input("How tall are you in meters? "))

print("Age:", age)
print("Height:", height, "meters")

# Calculate something
age_in_months = age * 12
print("You are approximately", age_in_months, "months old")
How old are you? 15 How tall are you in meters? 1.68 Age: 15 Height: 1.68 meters You are approximately 180 months old

Example 5: Simple Calculator

print("Simple Calculator")
print("=================")

num1 = float(input("Enter first number: "))

sum_result = num1 + num2

Simple Calculator ================ Enter first number: 12.5 Enter second number: 3.2 Results: Sum: 15.7 Difference: 9.3 Product: 40.0 Quotient: 3.90625

Example 6: Converting Numbers to Strings

age = 16
score = 95.5

# Convert numbers to strings for concatenation
message = "I am " + str(age) + " years old"
score_text = "My test score was " + str(score) + "%"

print(message)
print(score_text)

# Alternative using print() with commas (easier!)
print("I am", age, "years old")
print("My test score was", score, "%")
I am 16 years old My test score was 95.5% I am 16 years old My test score was 95.5 %

Exercises

  • Personal Greeting: Ask for the user's name and age. Print a personalized greeting like "Hello [name], you are [age] years old!"
  • Favorite Number Math: Ask the user for their favorite number. Convert it to an integer, then print the number multiplied by 2, and the number squared.
  • Rectangle Calculator: Ask the user for a rectangle's length and width (as floats). Calculate and display the area and perimeter.
  • Temperature Converter: Ask for a temperature in Celsius. Convert it to Fahrenheit using the formula: F = C * 9/5 + 32. Display both temperatures.
  • Age Calculator: Ask for the user's birth year. Calculate their current age (assume current year is 2024) and display it.
  • Simple Shopping: Ask for the price of an item and the quantity. Calculate the total cost and display it.
  • Circle Calculator: Ask for a circle's radius. Calculate the area (π × r²) and circumference (2 × π × r). Use 3.14159 for π.
  • Time Converter: Ask for a number of minutes. Convert and display the equivalent in hours and minutes (e.g., 90 minutes = 1 hour and 30 minutes).
  • BMI Calculator: Ask for height in meters and weight in kg. Calculate BMI using: BMI = weight / (height²). Display the result.
  • Student Report Card: Ask for the student's name and three test scores. Calculate the average score and create a formatted report showing the student's name, all three scores, and their average.

Common Mistakes to Avoid

❌ Don't do this:

age = input("Age? ")
next_year = age + 1        # Error: can't add int to string

number = int("hello")      # Error: "hello" isn't a number

float("3.14.15")          # Error: invalid decimal format

✅ Do this instead:

age = int(input("Age? "))  # Convert to int first
next_year = age + 1        # Now this works!

# Make sure input can be converted
number = int("25")         # This works fine

# Use proper decimal format
number = float("3.14")     # This works fine

Key Takeaways

  • input() always returns a string, even if the user types numbers
  • Use int() to convert strings to whole numbers
  • Use float() to convert strings to decimal numbers
  • Use str() to convert numbers to strings
  • You can convert input directly: age = int(input("Age? "))
  • Always make sure the input can be converted to avoid errors
