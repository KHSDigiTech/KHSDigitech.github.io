What are Loops?

Loops allow you to repeat code multiple times without writing it over and over. Instead of copying and pasting the same code, you can tell Python "do this 10 times" or "keep doing this until something changes."

Real-World Example

Think about brushing your teeth: "Repeat moving the toothbrush up and down 20 times." In programming: "Repeat printing 'Hello' 5 times."

Loop Flow

Start → Check condition → Run code → Go back to start

When condition becomes false → Stop and continue

For Loops

For loops are used when you know exactly how many times you want to repeat something.

Without Loops (Bad!)

print("Hello") print("Hello") print("Hello") print("Hello") print("Hello")

With For Loop (Good!)

for i in range(5): print("Hello")

The range() Function

The range() function creates a sequence of numbers that for loops can use:

Range Patterns:

  • range(5) → 0, 1, 2, 3, 4 (5 numbers starting from 0)
  • range(1, 6) → 1, 2, 3, 4, 5 (from 1 to 5)
  • range(2, 11, 2) → 2, 4, 6, 8, 10 (from 2 to 10, step by 2)

print("Counting to 5:") for i in range(1, 6): print("Number:", i) print("Done counting!")

You guessed it in {guesses} tries!") break # Exit the loop early elif guess < secret_number: print("Too low!") else: print("Too high!") remaining = max_guesses - guesses if remaining > 0: print(f"You have {remaining} guesses left.") if guess != secret_number: print(f"Sorry! For vs While Loops

Use FOR loops when:

  • You know exactly how many times to repeat
  • You want to do something for each item in a sequence
  • You're counting or iterating through numbers
# Print numbers 1 to 10 for i in range(1, 11): print(i)

Use WHILE loops when:

  • You don't know how many times to repeat
  • You want to continue until something specific happens
  • The condition might never become true
# Keep asking until correct answer = "" while answer != "yes": answer = input("Ready? ")

Common Mistakes to Avoid

⚠️ Infinite Loops

Make sure your while loop condition will eventually become false, or your program will run forever!

# BAD - This will run forever! count = 1 while count < 10: print(count) # Forgot to increment count! # GOOD - This will stop count = 1 while count < 10: print(count) count = count + 1 # Don't forget this!

❌ Don't do this:

for i in range(5) print(i) # Missing colon for i in range(5): print(i) # Missing indentation

✅ Do this instead:

for i in range(5): # Don't forget the colon print(i) # Proper indentation

Key Takeaways

  • For loops are great when you know how many times to repeat
  • While loops are perfect when you need to repeat until a condition changes
  • range() function is essential for for loops: range(start, stop, step)
  • Always make sure while loops have a way to stop (avoid infinite loops)
  • You can combine loops with if statements for powerful logic
  • Nested loops (loops inside loops) let you create patterns and tables
