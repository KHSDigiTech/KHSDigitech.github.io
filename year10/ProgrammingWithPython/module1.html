Module 1: Print Statements

Week 1 - Your First Python Code

Learning Objectives

By the end of this module, you will be able to:

  • Use the print() function to display text and numbers
  • Understand string literals and how to format text
  • Use escape characters for special formatting
  • Print multiple items in one statement
  • Control spacing and separators in output

What is the print() Function?

The print() function is Python's way of displaying information on the screen. It's like telling the computer to "show this to the user." Everything you want to display goes inside the parentheses.

Key Concept: Function Syntax

Functions in Python always have parentheses ( ) after their name. The information you want to give to the function goes inside these parentheses.

Basic Print Syntax

Here's the simplest way to use print:

print("Hello, World!")
Hello, World!

Important Notes:

  • Text must be surrounded by quotation marks (either "text" or 'text')
  • The parentheses are required
  • Python is case-sensitive - it must be print, not Print

Detailed Examples

Example 1: Printing Different Types of Text

print("Welcome to Python programming!")
print('Single quotes work too')
print("You can print numbers: 42")
print("And even decimals: 3.14")
Welcome to Python programming!
Single quotes work too
You can print numbers: 42
And even decimals: 3.14

Example 2: Printing Numbers (Without Quotes)

print(25)
print(3.14159)
print(-10)
25
3.14159
-10

Text vs Numbers

Text needs quotes, but numbers don't. print("25") prints the text "25", while print(25) prints the number 25. They look the same when printed, but Python treats them differently.

Example 3: Printing Multiple Items

print("My name is", "Sarah")
print("I am", 15, "years old")
print("Pi equals approximately", 3.14)
My name is Sarah
I am 15 years old
Pi equals approximately 3.14

Example 4: Empty Lines and Formatting

print("First line")
print()  # This prints an empty line
print("Third line")
print("Line 1\nLine 2\nLine 3")  # \n creates new lines
First line

Third line
Line 1
Line 2
Line 3

Example 5: Escape Characters

print("She said, \"Hello there!\"")  # \" for quotes inside quotes
print("Tab\tspaced\ttext")          # \t for tab spacing
print("C:\\Users\\Documents")        # \\ for backslash
She said, "Hello there!"
Tab    spaced    text
C:\Users\Documents

Exercises

  • Basic Greeting: Write a print statement that displays: Hello, my name is [your name]
  • Your Information: Print your age as a number (without quotes) on a separate line.
  • School Subject: Print: My favorite subject is Digital Technology
  • Multiple Items: Print your first name and last name in one print statement, separated by a space.
  • Numbers and Text: Print: There are 365 days in a year using both text and a number in the same print statement.
  • Empty Line: Print "First line", then an empty line, then "Last line" (use 3 print statements).
  • Quotes Inside Text: Print: The teacher said, "Good work!"
  • New Lines: Print "Line 1", "Line 2", and "Line 3" each on separate lines, but use only ONE print statement with \n.
  • Tab Spacing: Print: Name: John Age: 16 using \t for the spacing.
  • Creative Output: Create a simple "business card" for yourself using print statements. Include your name, school year, and favorite hobby. Make it look neat using spaces, tabs, or new lines.

Common Mistakes to Avoid

❌ Don't do this:

Print("Hello")      # Capital P - Python is case sensitive
print(Hello)        # Missing quotes around text
print "Hello"       # Missing parentheses
print('Hello")      # Mismatched quotes

✅ Do this instead:

print("Hello")      # Correct!
print("Hello")      # Text needs quotes
print("Hello")      # Parentheses required
print("Hello")      # Matching quotes
