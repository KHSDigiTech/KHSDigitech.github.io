Assessment Task

2-Week Programming Project - Weeks 7 & 8

Project Overview

For your final assessment, you'll create a complete Python program that demonstrates all the concepts you've learned throughout this course. This project will take place over two weeks and will be your opportunity to showcase your programming skills.

Assessment Goals:

  • Apply all programming concepts from Modules 1-6
  • Create a functional, interactive program
  • Demonstrate problem-solving and programming logic
  • Write clean, well-commented code
  • Present your work to the class

Project Options

Choose ONE of the following project options:

Option 1: Text Adventure Game

Create an interactive story-based game where players make choices that affect the outcome.

Features to include:

  • Story with multiple paths and endings
  • Player choice system using input and if statements
  • Character stats (health, items, score)
  • Inventory system using lists
  • Save/load game progress

Option 2: Personal Finance Calculator

Build a comprehensive calculator to help manage personal finances and budgeting.

Features to include:

  • Income and expense tracking
  • Budget planning and recommendations
  • Savings goal calculator
  • Data persistence (save/load financial data)
  • Financial reports and summaries

Option 3: Quiz Game System

Create an interactive quiz game with multiple categories and difficulty levels.

Features to include:

  • Multiple quiz categories (science, math, history, etc.)
  • Different difficulty levels
  • Score tracking and high scores
  • Question bank stored in lists
  • Timer functionality (bonus feature)

Option 4: Student Grade Manager

Develop a system to manage student information and calculate grades.

Features to include:

  • Add/remove students and courses
  • Input and modify grades
  • Calculate GPAs and course averages
  • Generate report cards
  • Search and filter functionality

Technical Requirements

Your project MUST demonstrate ALL of the following concepts:

1. Print Statements & Output Formatting

Use print statements for user interaction, menus, and displaying information in a clear, well-formatted way.

2. Variables & Data Types

Use appropriate variable names and data types (strings, integers, floats, booleans) throughout your program.

3. User Input & Type Conversion

Get input from users and convert data types when necessary. Include input validation to handle errors.

4. If Statements & Decision Making

Use if, elif, and else statements to create branching logic and respond to different user choices.

5. Loops

Use both for and while loops appropriately. Include a main program loop to keep the program running.

6. Lists & Functions

Create and use lists to store data. Write at least 3 custom functions that make your code more organized and reusable.

7. Code Quality

Write clean, readable code with meaningful variable names, proper indentation, and helpful comments explaining your logic.

Project Timeline

Week 7 - Planning & Development

  • Day 1: Choose project option and create detailed plan
  • Day 2: Begin coding - focus on basic structure and main menu
  • Day 3: Implement core features and user input handling
  • Day 4: Add data storage (lists) and core functions
  • Day 5: Continue development and begin testing

Week 8 - Completion & Presentation

  • Day 1: Complete all required features
  • Day 2: Add error handling and polish user interface
  • Day 3: Final testing and bug fixes
  • Day 4: Prepare presentation and documentation
  • Day 5: Project presentations to class

Success Tips

  • Start Simple: Get a basic version working first, then add features
  • Test Frequently: Run your code often to catch errors early
  • Use Functions: Break your code into smaller, manageable functions
  • Handle Errors: Use try-except blocks for user input validation
  • Comment Your Code: Explain what each section does
  • Ask for Help: Don't hesitate to ask questions if you're stuck
  • Plan Before Coding: Spend time thinking about the structure before writing code

Submission Requirements

What to Submit:

  • Complete Python (.py) file with your program
  • Brief written explanation (1-2 paragraphs) of your project
  • List of any challenges you faced and how you solved them
  • Ideas for future improvements or features

Presentation Requirements:

  • 5-minute demonstration of your program
  • Explain your code structure and key functions
  • Show all required programming concepts in action
  • Discuss challenges and solutions
